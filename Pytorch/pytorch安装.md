# pytorch安装/升级
进入pytorch官网： https://pytorch.org/
选择自己对应安装的版本:

Linux Python3.5 No-CUDA
![](../__pics/pytorch安装.png)

或者直接
```shell
$ sudo pip3 install torchvision
```
![](../__pics/torchvision.png)

升级的时候,后面是官网查到的对应版本
```shell
$ sudo pip3 install --upgrade torch-0.3.0.post4-cp27-cp27mu-linux_x86_64.whl
```
